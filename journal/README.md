# Journal

The "how we got here" log.

This folder is NOT documentation. It's the **record of our thinking** — dated entries that capture the questions we asked, the tensions we surfaced, the options we considered, and the decisions we made (and deferred). When future-you wonders *why* we picked X over Y, the answer lives here, not in the design docs.

Design docs tell you **what** we decided. The journal tells you **why**.

## Naming convention

`YYYY-MM-DD-short-slug.md`
